--- layout: reference title: map category: manipulation blurb: | `map` is a function associated with the `Wave` class that lets you apply any function you want to transform a wave. #### Usage - `wave.map(() => 0)` fills a wave with zeroes - `wave.map(() => Math.random())` fills a wave with random numbers - `wave.map(e => e * e)` squares the output of a wave - `wave.map(e => Math.abs(e))` returns the absolute value of a wave - `wave.map((e, i) => i / 128)` returns a wave whose entries linearly increase from 0 to 1 `map` allows you to build any effect you like. All the other wave manipulation functions are built using this function.
